# Madhav Ecommerce Sales Dashboard

## Project Overview
This project presents a Power BI dashboard created to analyze ecommerce sales data from the Madhav store. The dashboard provides insights into sales performance, profit trends, customer behavior, and product category distribution.

## Tools Used
- Microsoft Power BI
- Power Query
- CSV Dataset

## Dataset
The project uses two datasets:

Orders.csv
Contains order details such as customer name, state, and order ID.

Details.csv
Contains product details including amount, profit, quantity, category, and payment mode.

## Dashboard Insights
The dashboard provides insights including:

- Total Sales: 438K
- Total Profit: 37K
- Total Quantity Sold: 5615
- Category-wise sales distribution
- Profit analysis by month
- Payment mode usage
- State-wise sales performance

## Dashboard Visualizations
The dashboard includes:

- Sales by State
- Profit by Month
- Quantity by Category
- Payment Mode Distribution
- Profit by Sub-Category
- Customer Sales Contribution

## Conclusion
The Power BI dashboard helps convert raw sales data into meaningful insights, helping businesses understand sales trends and customer behavior for better decision making.
